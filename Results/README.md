This directory contains the results from testing obtained for this project

**Hardware Validation Proof:**
* **`Utilization_Report.txt`**: Confirms the design uses 10,060 LUTs (48.4%), leaving over 50% of the fabric available for future PCIe/10GbE integration.
* **`Power_Report.txt`**: Confirms total on-chip power is just 0.246W.
* **`Timing_Summary.txt`**: **Note on printed poster:** While the physical poster displays a Worst Negative Slack (WNS) of +0.183ns from an earlier synthesis run, this final report proves the optimized hardware achieved a closed timing of **+0.294ns** at 100 MHz.
